    Common Questions

    Is this a replacement for QuickBooks, Xero, or FreshBooks?

    No. This is a management system that sits alongside QuickBooks, Xero, or whatever bookkeeping tool you use. It gives you operational visibility—pipeline, project profitability, client LTV—that accounting software doesn’t provide. You still need your accountant for tax filing.

    How long does it take to set up?

    Plan for 2–3 hours to enter your existing clients, projects, and financial data. After initial setup, daily maintenance is 5–10 minutes for time logging and expense entry.

    Does it include a chart of accounts?

    The system uses standard income/expense categories suitable for most service businesses. If you use a specific COA structure, you can adapt the category names in the setup sheet.

    Can multiple people use it?

    It’s a single Excel workbook. Multiple team members can use it if they have access to the file (OneDrive/SharePoint), but it’s not a multi-user database. Best for solo operators and very small teams.
